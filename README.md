# 구현 예정
- CQRS 패턴 적용을 통한 Multi process 환경 구성
- Multi Thread를 기반으로 한 Spring batch 형성
- 서비스 운영을 위한 모니터링 서비스 구축
- Grafana를 통해 Prometheus 데이터 시각화
- 부하테스트를 통한 서비스 수준 목표(SLO) 작성
- 부하테스트 툴을 통해 각 서비스들의 현재 지표 파악
